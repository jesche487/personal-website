# Personal Website

This is a simple static personal website for Jesse Chen. It is built using plain HTML and CSS and is organized for easy editing and friendly handoff.

## Project structure

- `index.html` — main homepage with hero, about, experience, projects, skills, and contact sections.
- `fun.html` — a personal interests page for hobbies, a cappella content, and other non-work items.
- `resume.html` — a redirect page that sends visitors to the PDF resume.
- `style.css` — shared site styling used across all pages.
- `assets/` — stores non-HTML assets:
  - `me.jpg` — your profile photo.
  - `Jesse Chen Resume 10-2025.pdf` — your resume document.
- `Personal Website.code-workspace` — VS Code workspace file for this project.

## Notes

- All pages use the same `style.css` and Google font import.
- The website is fully static and can be opened directly in a browser.
- `resume.html` uses a redirect so users are forwarded to the resume PDF.

## Editing tips

- To update your homepage content, edit `index.html`.
- To add or change personal details, edit `fun.html`.
- To change styling, edit `style.css`.
- If you add new images or documents, put them in `assets/` and update links accordingly.

## Local preview

Open `index.html` in your browser to preview the site locally.
